Pillar 1

Identity: Unity Catalog as the Foundation

  • UC governs models, agents, functions, connections, and data
  • EXECUTE grants control who can deploy custom models and agents
  • Foundation Model Permissions (Preview): system.ai makes every FMAPI model a UC securable
  • REVOKE EXECUTE = model disabled org-wide across PPT, PT, and Batch
  • Secret scope ACLs govern API keys for external model providers
  • One identity model across all workspaces in the account
Pillar 2

Runtime: AI Gateway v2

Guardrails

  • Safety filtering (Llama Guard)
  • PII detection (Presidio)
  • Content moderation
  • Dry-run mode for testing

Controls

  • Rate limits: endpoint, user, SP, group
  • Fallback routing across providers
  • Traffic splitting for A/B testing
  • MCP server governance

Logging

  • Inference tables: full payload to UC Delta
  • Usage tracking with cost attribution

API Coverage

  • Native: OpenAI, Anthropic, Gemini
  • Custom API for any external service
Pillar 3

Observability: See Everything

  • System tables (account-level): all endpoints, all workspaces, one query
  • MLflow 3 production monitoring: scorers run on live traces continuously
  • 14 built-in LLM judges + multi-turn conversation judges
  • Third-party scorers: DeepEval, RAGAS, Guardrails AI in one evaluate() call
  • UC OTel tracing: span-level traces stored directly in UC Delta
  • Genie space: CISO asks questions in natural language
Pattern

Policy as Data

  • Guardrail policy stored in UC Delta table
  • Governed by UC ACLs (only admins can modify)
  • Versioned, auditable, queryable via SQL
  • Same policy enforced across all workspaces

For the CISO

What the CISO Sees

Account Estate

Every endpoint by type, by workspace. Models, agents, providers at a glance.

Safety Block Rate

The key metric: blocked requests / total requests. Per endpoint, per workspace, over time.

Threat Insights

What triggered blocks: PII in prompts, jailbreak attempts, unsafe instructions, secrets.

Requester Analysis

Who is calling what, token consumption, error rates per user.

Compliance Status

Which endpoints are governed, which aren't. Tamper attempts and auto-remediation log.

Genie Space

Natural language queries: "Show me all endpoints without guardrails."

Continuous Quality

MLflow Scorers: Dev to Prod

Zero-cost (every trace)

  • PII detection (regex-based)
  • Secret detection (API keys, AWS keys)
  • Jailbreak detection (pattern matching)
  • BLEU/ROUGE scores (RAGAS)

LLM-based (sampled)

  • Safety, Groundedness, Guidelines
  • DeepEval: Relevancy, Faithfulness, Toxicity
  • Multi-turn conversation judges
  • Custom judges via make_judge()

Same scorers in dev (mlflow.genai.evaluate()) and prod (production monitoring). Assessments archived to UC Delta.
